A _project_ in the Prisma Data Platform is an organizational unit for your JavaScript/TypeScript application that uses Prisma. You use projects to manage:

* environments
    * database connection
    * Data Proxy region
    * GitHub repository and branch
* [members](/data-platform/members)
* [paid plan and billing](/data-platform/billing/plans-and-quotas)

When you create or add a new project in the Prisma Data Platform, you must define a database connection string for your application, a GitHub repository where your application is stored, and a geographic region for the Data Proxy so that it is closer to region of your database.
