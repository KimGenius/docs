<TopBlock>

In the project settings, you can change the project display name and the project handle.
</TopBlock>

#

**Prerequisites and considerations**

You must be the [Project owner or have the Admin role](/data-platform/members/user-roles) to edit the project settings.

**Steps**

1. In the upper-left, click your project handle and select **Project settings**.
    <img
        src="https://res.cloudinary.com/prismaio/image/upload/v1666280499/docs/pdp/manage-project-members/manage-project-members-01-open-project-settings_zaodgg.png"
        alt="Prisma Data Platform - Manage project members - Open project settings"
        width="350px"
    />
2. In the **Project Name** text box, change the project name.
    <Admonition>
    
    **Note**<br /><br />

    The project name appears in several places in the Prisma Data Platform.<br /><br />
          &nbsp;• [_Projects_](https://cloud.prisma.io/projects) page<br />
          &nbsp;• [_Billing_](https://cloud.prisma.io/billing) page<br />
          &nbsp;• **Projects** drop-down in the upper-left
    </Admonition>
3. In the **Project Handle** text box, change the project handle.
    <Admonition>
    
    **Note**<br /><br />

    The project handle defines the project URL in the Prisma Data Platform.<br /><br />

    `https://cloud.prisma.io/projects/<project-handle>`<br /><br />

    For example, if your project handle is `nextjs-db`, your project URL is `https://cloud.prisma.io/projects/nextjs-db`
    </Admonition>

**Result**

The changes to the project name and handle are applied automatically.
